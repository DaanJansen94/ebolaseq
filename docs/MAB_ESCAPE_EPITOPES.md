## Reference baseline

Epitope positions use **UniProt Q05320 numbering**. Amino acids are the **2014 West African Makona** reference (GenBank [KJ660346.2](https://www.ncbi.nlm.nih.gov/nuccore/KJ660346.2)), matching the GP used in REGN-EB3 escape work; at all monitored sites these match Mayinga-76 (Q05320).
